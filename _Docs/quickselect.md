# Quickselect

<p>
  <a href="https://leetcode.com/problemset/">

Quickselect partitions data like quicksort but only recurses into the side containing the desired rank. It is useful for finding kth smallest, kth largest, and median-like elements. <br>
<br>
This section collects LeetCode problems tagged with Quickselect. Use it as a focused practice set: start with the easier rows to build the core pattern, then move into medium and hard problems to see how the idea combines with other techniques. <br>
