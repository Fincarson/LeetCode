Bucket sort groups values into buckets based on value ranges or frequencies. It is useful when the input has exploitable bounds or when grouping avoids a full comparison sort. <br>
<br>
This section collects LeetCode problems tagged with Bucket Sort. Use it as a focused practice set: start with the easier rows to build the core pattern, then move into medium and hard problems to see how the idea combines with other techniques. <br>
